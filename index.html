Git

Trabalhando em Equipe com Controle e Segurança dos ativos de TI

Um curso que explora os benefícios de utilizar o Git como ferramenta de controle de versão para projetos em qualquer linguagem, em qualquer plataforma.

Principais benefícios para esse sistema do corregedor geral da sessão do tribunal:

  • Funciona de maneira distribuída para o sessão judiciaria
  • Permite a edição concorrente de arquivos do projeto
  • Não depende de uma conexão ativa com um servidor com o PC do cliente
  • Varias maneira de conluir o projeto
  • Linux traz mais estabilidade ao ambiente computacional da sessão judicial
